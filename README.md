# Umbria
## О проекте
Umbria - это система автоматизированные аренда зонтов для университетов, кампусов и общественных пространств.

Проект решает проблему внезапной непогоды: пользователь может быстро взять зонт в одной точке и вернуть его в другой.

----
## Основные возможности
- Поиск ближайших станций с зонтами
- Авторазиция по пропуску / аккаунту
- Аренда и возврат зонта
- Учёт пользователей и активных аренд
- Отслеживание статуса ячеек(свободна / занята)
----
## Архитектура проекта
Система включает:
- Backend - обработка логики аренда
- База данных пользователей, зонтов и операций
- API - взаимодействие клиента и пользователей
- (Опциональна) Frontend - интерфейс пользователя
----
## 🗄 Структура базы данных

### Пользователи (Users)
- id  
- имя  
- RFID / пропуск  
- баланс  
- статус  

### Зонты (Umbrellas)
- id  
- уникальный код (RFID / метка)  
- статус (свободен / выдан)  
- текущая локация  

### Станции (Stations)
- id  
- название  
- местоположение  

### Ячейки (Slots)
- id  
- station_id  
- статус (свободна / занята)  

### Аренда (Rentals)
- id  
- user_id  
- umbrella_id  
- время выдачи  
- время возврата  
- статус  

---

## ⚙️ Стек технологий

Пример:

- Backend: Django 
- БД: SQLite
- API: REST
- Дополнительно:
  - RFID 
  - WebSocket (для обновлений в реальном времени)

## 🛠 Установка и запуск


### Клонировать репозиторий
git clone https://github.com/m3wqd/Umbria.git

cd Umbria

### Установить зависимости
pip install -r requirements.txt

### Применить миграции
python manage.py migrate

### Запуск сервера
python manage.py runserver
